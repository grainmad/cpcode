    * /a-problem-everyday 每天一道codeforces，由脚本生成，脚本可指定题目难度与标签。
    * /contest 比赛，脚本cf_gen_sol.py用于生成题解模板。
* /data cf的题目数据
* /leetcode leetcode上一些题目。
* /template 一些算法模板
    * /viscode/linux 配置了linux环境下一些脚本的别名。

* hexo-deployer.py 将指定文件夹下的题解生成为hexo博客文章，并自动部署更新hexo博客。
